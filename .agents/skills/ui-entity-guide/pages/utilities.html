🔐 Auth — Xác thực & Phân quyền

AdminAuthService quản lý đăng nhập, user hiện tại, và phân quyền.

👤 Lấy user hiện tại
// Inject AdminAuthService
constructor(private authen: AdminAuthService) {}

// Lấy user object (AdminUserDto)
let user = this.authen.account;
let name = user.FullName;
let email = user.Email;
let isAdmin = user.IsAdmin;
let id = user.Id;
🔑 Đăng nhập / Đăng xuất
// Đăng nhập
let result = await this.service.signin({ Email: email, Password: pwd });
if (ResultApi.IsSuccess(result)) {
  let account = result.Object as AdminUserDto;
  await this.authen.login(account, rememberMe);
}

// Đăng xuất
await this.authen.logout();

// Khóa màn hình
await this.authen.lock();
🛡️ Kiểm tra quyền
// Kiểm tra quyền async
let allow = await this.authen.permissionAllow('category', 'Edit');

// Kiểm tra quyền sync (nhanh, dùng cache đã load)
let allow = this.authen.permissionAllowSync('category', 'Delete');

// Lọc actions theo quyền
let allowedActions = await this.authen.actionsAllow(
  CategoryEntity, this.obj.Actions
);
HàmMô tả
accountGetter — lấy AdminUserDto hiện tại (Id, Email, FullName, IsAdmin, Token...).
login(account, rememberMe?)Lưu session + redirect đến /admin.
logout(navigate?)Xóa session + redirect đến /admin/signin.
lock()Khóa màn hình, redirect đến /admin/lock.
permissionAllow(controller, action?)Async — kiểm tra quyền. IsAdmin → luôn true.
permissionAllowSync(controller, action?)Sync — dùng cache đã load. Nhanh hơn.
actionsAllow(reference, actions)Lọc ActionData[] theo quyền.
loadAuthenData()Load permissions + link permissions.
ℹ️account đọc từ sessionStorage hoặc localStorage (tùy rememberMe). Token được lưu cùng với user object.

🌐 i18n — Đa ngôn ngữ

Dùng TranslatePipe trong template hoặc AdminTranslateService trong code.

📝 Trong Template (HTML)
<!-- Pipe translate -->
<span>{{ 'General.Save' | translate }}</span>
<button>{{ 'General.Cancel' | translate }}</button>
<label>{{ 'Category.Name' | translate }}</label>
⚙ Trong Code (TypeScript)
// Inject translate service (GridComponent / EditComponent đã có sẵn)
let text = this.translate.transform('General.Confirm');
let label = this.translate.transform('Category.Name');

// Hoặc dùng UtilityExHelper (static)
let text = UtilityExHelper.transformText('Core.Error');
📋 Key format
KeyVí dụ
General.*General.Save, General.Cancel, General.Ok, General.Close, General.Confirm
Core.*Core.Error, Core.Warning, Core.Success, Core.Loading, Core.Notify
{Module}.*Category.Name, Category.Code — key cho từng module.
⚠️Ngôn ngữ được lưu trong localStorage với key AppConfig.LanguageKey. Thay đổi ngôn ngữ cần reload page.

📡 Events — Event Bus (AdminEventService)

AdminEventService cung cấp EventEmitter cho giao tiếp giữa các component.

EventTypeMô tả
RefreshGridsTableDataYêu cầu grid reload dữ liệu.
RefreshUpdateGridsstringYêu cầu grid cập nhật theo tên.
RefreshItemsRefreshOptionItemRefresh items trên dropdown/combobox.
RefreshMenusanyRefresh sidebar menu.
RefreshItemDetailanyRefresh chi tiết item.
SignalrNotifyanyNhận notification từ SignalR.
ValidateObjectExTrigger validation cho 1 property.
ResetValidateObjectExReset validation cho 1 property.
AutoUpload{ decorator, object }Tự động upload file/image/video.
HideComboboxanyẨn tất cả dropdown đang mở.
TextChangeRefreshOptionItemText thay đổi trên editor.
💡 Ví dụ — Emit & Subscribe
// Emit — refresh grid
this.eventService.RefreshGrids.emit(this.itemData);

// Subscribe — lắng nghe SignalR notification
this.eventService.SignalrNotify.subscribe((data) => {
  // xử lý notification realtime
});

// Refresh dropdown items
this.eventService.RefreshItems.emit({
  property: 'CategoryId',
});
ℹ️Grid/Edit component đã inject eventService sẵn. Chỉ cần this.eventService.XXX.emit().
