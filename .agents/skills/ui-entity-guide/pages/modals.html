📦 Tổng quan hệ thống Modal

Tất cả modal được quản lý qua AdminDialogService. Service emit event → Modal component lắng nghe theo DialogType.

🗂 Danh sách Modal Components

ModalConfirmComponentConfirmXác nhận Có/Không. Hỗ trợ async + nút Result.
ModalPromptComponentPromptNhập liệu text. Trả về giá trị qua callback.

ModalChangePasswordComponentChangePasswordĐổi mật khẩu.
📐 ModalSizeType
export enum ModalSizeType {
  Small     = 1,   // ~400px
  Medium,          // ~600px
  Large,           // ~800px
  ExtraLarge,      // ~1100px
  FullScreen,      // 100% viewport
}
ℹ️Khi màn hình < 1400px, popup tự tăng 1 size (Small → Medium, Medium → Large...).

💬 Dialog — Alert / Confirm / Prompt

Các hàm gọi trực tiếp trên this.dialogService (inject AdminDialogService).

1️⃣ Alert — Thông báo
// Thông báo đơn giản
this.dialogService.Alert('Tiêu đề', 'Nội dung thông báo');

// Thông báo từ kết quả API
this.dialogService.AlertResult('Lỗi', result);

// Thông báo tự đóng sau N giây
this.dialogService.AlertTimeOut('Tiêu đề', 'Nội dung', 10);

// Shortcut
this.dialogService.Success('Thành công!');
this.dialogService.Error('Có lỗi xảy ra');
this.dialogService.Warning('Cảnh báo!');
this.dialogService.Message('Thông tin');
2️⃣ Confirm — Xác nhận
// Confirm sync
this.dialogService.Confirm('Bạn chắc chắn?', () => {
  // OK
}, () => {
  // Cancel (optional)
});

// Confirm async
this.dialogService.ConfirmAsync(
  'Xóa mục này?',
  async () => { await this.service.delete(id); },
  null,          // cancelFunction
  'Xác nhận',    // title
  'Đồng ý',     // confirmText
);
HàmParamsMô tả
Confirmcontent, okFn, cancelFn?, title?Xác nhận sync.
ConfirmAsynccontent, okFnAsync, cancelFn?, title?, confirmText?, resultText?, resultFn?Xác nhận async. Hỗ trợ nút Result thứ 3.
ConfirmRedirecttitle, content, okFnXác nhận rồi redirect.
3️⃣ Prompt — Nhập liệu
this.dialogService.PromptAsync(
  'Nhập tên mới',     // label
  'Giá trị mặc định', // object (giá trị ban đầu)
  async (name) => {
    // name = giá trị user nhập
    await this.rename(name);
  },
  null,              // cancelFunction
  'Đổi tên',         // title
  'Lưu',             // confirmText
);
4️⃣ Loading — Đang tải
let dialog = this.dialogService.Loading('Đang xử lý', 'Vui lòng đợi...');
// ... xử lý xong
this.dialogService.HideDialog(dialog);
⚠️Loading có restrict: true → user không thể đóng. Phải gọi HideDialog hoặc HideAllDialog.

🪟 Wrapper — Popup chứa Component

WapperAsync inject Angular component vào popup. Component được truyền data qua params và cần implement confirm() / reject().

📐 Cách gọi WapperAsync
this.dialogService.WapperAsync({
  title: 'Tiêu đề popup',
  object: MyComponent,         // Component để inject
  objectExtra: { id: 123 },    // Data truyền vào component
  size: ModalSizeType.Large,
  confirmText: 'Lưu',
  cancelText: 'Đóng',
  rejectText: 'Từ chối',       // nút thứ 3 (optional)
  resultText: 'Kết quả',       // nút thứ 4 (optional)
}, async (result) => {
  // okFunction — sau khi component.confirm() trả true
}, async (result) => {
  // rejectFunction
}, async (result) => {
  // resultFunction
}, () => {
  // cancelFunction
});
📋 DialogData params
ParamTypeMô tả
objectComponentAngular Component class để render.
objectExtraanyData truyền vào component.params.
titlestringTiêu đề popup.
descriptionstringMô tả phụ dưới tiêu đề.
sizeModalSizeTypeKích thước popup.
confirmTextstringText nút xác nhận. null → ẩn.
cancelTextstringText nút đóng.
rejectTextstringText nút Reject (nút thứ 3).
resultTextstringText nút Result (nút thứ 4).
classNamestringCSS class thêm vào popup.

⚙ Component bên trong popup cần implement
export class MyComponent {
  @Input() params: any;   // nhận objectExtra từ popup

  ngOnInit() {
    let id = this.params['id'];  // hoặc this.getParamPopup('id')
  }

  // Wrapper gọi khi bấm Confirm
  async confirm(): Promise<boolean> {
    let valid = await validation(this.item);
    if (!valid) return false;   // popup không đóng
    await this.service.save(this.item);
    return true;               // popup đóng + gọi okFunction
  }

  // Wrapper gọi khi bấm Reject (optional)
  async reject(): Promise<boolean> { return true; }

  // Wrapper gọi khi bấm Result (optional)
  async result(): Promise<boolean> { return true; }

  // Wrapper gọi khi đóng popup (optional)
  close() { }
}
ℹ️Quan trọng: confirm() trả false → popup không đóng (dùng cho validate). Trả true → đóng + gọi okFunctionAsync.
📑 WapperAboveAsync — Popup lớp 2

Cùng API với WapperAsync, nhưng mở popup chồng lên popup hiện tại.

this.dialogService.WapperAboveAsync({
  title: 'Chọn nhân viên',
  object: ChoiceUserComponent,
  size: ModalSizeType.Large,
}, async (result) => { });
⚠️HideAllDialog() đóng tất cả popup (cả Wrapper + WrapperAbove). HideDialog(dialog) chỉ đóng 1 popup cụ thể.

⚡ Auto — Tự sinh form từ Entity

DialogType.Automatic tự render form từ Entity decorators. Không cần viết component riêng. Dùng qua popupAutoAsync() trên Grid/Edit.

📐 DialogAutoData
ParamTypeMô tả
titlestringTiêu đề popup.
sizeModalSizeTypeKích thước popup.
objectValue.referenceEntity classEntity dùng để tự sinh form (đọc decorator).
objectValue.properties(string | ObjectEx)[]Danh sách properties hiển thị.
objectValue.objectanyGiá trị mặc định (entity instance).
confirmTextstringText nút xác nhận.
confirmFunction(item) => PromiseCallback nhận entity object khi xác nhận.
cancelTextstringText nút đóng.
classNamestringCSS class bổ sung.
💡 Ví dụ — Thêm nhanh từ Grid
this.popupAutoAsync({
  title: 'Thêm danh mục',
  size: ModalSizeType.Medium,
  objectValue: {
    reference: CategoryEntity,
    properties: ['Code', 'Name', 'Description'],
    object: new CategoryEntity(),
  },
  confirmText: 'Lưu',
  confirmFunction: async (item) => {
    await this.service.save('category', item);
  },
});
💡 Ví dụ — Chỉnh sửa từ Edit form
let entity = await this.autoLoadItem(itemId, CategoryEntity);
this.popupAutoAsync({
  title: 'Sửa danh mục',
  objectValue: {
    reference: CategoryEntity,
    properties: ['Code', 'Name'],
    object: entity,  // dữ liệu có sẵn
  },
  confirmText: 'Cập nhật',
  confirmFunction: async (item) => {
    await this.service.update('category', item);
  },
});
ℹ️Khả dụng trên: GridComponent.popupAutoAsync(), EditComponent.popupAutoAsync(), EditComponent.popupAutoAboveAsync().

📤 Export — Xuất dữ liệu

ModalExportDataComponent render trong Wrapper. Hỗ trợ xuất Excel, CSV, PDF. Grid tự mở popup này khi user click Export trên toolbar.

LimitnumberGiới hạn số dòng. Mặc định: 1000.
DateRangeDate[]Phạm vi ngày lọc cho export.
PageSizenumberKích thước trang PDF.

💡 Gọi thủ công
this.popupAsync({
  title: 'Xuất Excel',
  object: ModalExportDataComponent,
  size: ModalSizeType.Medium,
  objectExtra: {
    Data: this.itemData,
    Type: ExportType.Excel,
    Reference: CategoryEntity,
    Limit: 5000,
  },
  confirmText: 'Xuất',
});
ℹ️Grid tự tạo popup Export khi khai báo Exports trong GridData. Thường không cần gọi thủ công.
