📦 CRUD — Thao tác dữ liệu cơ bản

Tất cả hàm trả về Promise<ResultApi>. Kiểm tra kết quả bằng ResultApi.IsSuccess(result).

HàmParamsAPI EndpointMô tả
saveobjName, obj, id?POST/PUT /admin/{objName}Tạo hoặc cập nhật. Tự detect POST (new) / PUT (update) theo obj.Id.
updateobjName, obj, columns[]PUT /admin/{objName}?columns=A,BCập nhật chỉ các cột chỉ định.
itemobjName, idGET /admin/{objName}/{id}Lấy 1 item theo ID.
itemstableData?, url?POST /admin/{name}/itemsLấy danh sách (paging, filter, sort).
deleteobjName, idDELETE /admin/{objName}/{id}Xóa vĩnh viễn.
trashobjName, idDELETE /admin/{objName}/trash/{id}Soft delete (chuyển vào thùng rác).
activeobjName, idPOST /admin/{objName}/active/{id}Toggle trạng thái active.
multipleDeleteobjName, ids[]DELETE /admin/{objName}?ids=1,2,3Xóa nhiều items.
multipleTrashobjName, ids[]DELETE /admin/{objName}/trash?ids=Soft delete nhiều items.
trashVerifyobjName, idDELETE /admin/{objName}/trashVerify/{id}Khôi phục từ thùng rác.
existsurl, property, objId?, value?GET /admin/{url}?property=&value=Kiểm tra tồn tại (dùng cho validate unique).
resetCacheobjNameGET /admin/{objName}/resetCacheReset cache server.
💡 Ví dụ
// Lưu entity (tự detect POST/PUT)
let result = await this.service.save('category', this.item);
if (ResultApi.IsSuccess(result)) { /* ok */ }

// Lấy 1 item
let result = await this.service.item('category', 123);
let entity = EntityHelper.createEntity(CategoryEntity, result.Object);

// Xóa + confirm
this.dialogService.ConfirmAsync('Xóa mục này?', async () => {
  await this.service.delete('category', item.Id);
  await this.loadItems();
});
ℹ️save() tự xóa IsActive, IsDelete, CreatedBy, CreatedDate... trước khi gửi. Không cần clean thủ công.

🔗 Custom API — Gọi API tùy chỉnh

HàmParamsMô tả
callApiUrlurl, obj?, method?Gọi API theo URL. Tự thêm /admin prefix. Method tự detect: có obj → POST, không → GET.
callApicontroller, action, obj?, method?Gọi /admin/{controller}/{action}.
callApiNonAdminUrlurl, obj?, method?Gọi API không có prefix /admin.
saveByUrlurl, obj, method?POST/PUT đến URL cụ thể (auto clean fields).
requestByUrlurl, method?, obj?Request raw theo URL + method.
filterItemsobjName, name, value, compare?Lọc items theo 1 trường (tự detect CompareType).
filterItemsByUrlurl, name, value, compare?Giống filterItems nhưng dùng URL tùy chỉnh.
💡 Ví dụ
// Gọi GET /admin/category/tree
let result = await this.service.callApiUrl('/category/tree');

// Gọi POST /admin/report/summary với body
let result = await this.service.callApi('report', 'summary', { Year: 2024 });

// Gọi API bên ngoài (không có /admin prefix)
let result = await this.service.callApiNonAdminUrl('/public/config');

// Lọc nhanh
let result = await this.service.filterItems('staff', 'DepartmentId', 5);
⚠️callApiUrl tự thêm /admin prefix. VD: callApiUrl('/category/tree') → GET /admin/category/tree.
📋 ResultApi — Xử lý kết quả
let result = await this.service.callApiUrl('/...');
if (ResultApi.IsSuccess(result)) {
  let data = result.Object;    // object trả về
  let items = result.Object;   // mảng items (cho /items)
  let total = result.ObjectExtra?.Total;
} else {
  this.dialogService.ErrorResult(result);
}

🔍 Lookup — Tìm kiếm nhanh

Dùng cho Dropdown, Autocomplete. Hỗ trợ lọc theo cột, phân trang, search text.

HàmParamsMô tả
lookupurl, columns?, by?, useQuery?, search?, pageIndex?, pageSize?Lookup GET. VD: /category/lookup.
lookupItemurl, value, key, columns?Lookup 1 item theo key=value.
lookupDateTimeurl, columns?, by?, search?, pageIndex?, pageSize?Lookup theo DateTime range.
columnLookupurl, tableData, columns?, by?, ...Lookup POST với TableData body.
popupItemstableData?, url?Items cho popup (GET, query params).
💡 Ví dụ
// Lookup danh sách cho dropdown
let result = await this.service.lookup(
  'category/lookup',
  ['Id', 'Name'],               // columns
  null,                         // by
  false,                        // useQueryParms
  'search text',               // search
);

// Lookup 1 item theo ID
let result = await this.service.lookupItem(
  'category/lookup', 5, 'Id', ['Name', 'Code']
);

📤 Upload — Tải file lên

HàmMô tả
upload(url, UploadData)Upload file qua XHR. Có callback: processFunction (%), completeFunction, failFunction.
uploadPercent(url, UploadData)Upload qua HttpClient với progress tracking.
autoUpload(objName, obj, props?)Tự detect Image/File/Video properties và upload. Dùng trước save().
customUpload(custom, UploadData)Upload đến server bên ngoài (custom headers, auth).
downloadFile(objName, tableData)Download export file (Excel/PDF/CSV).
💡 Ví dụ — autoUpload
// Auto upload trước khi save
let uploader = await this.service.autoUpload('product', this.item);
let result = await uploader.update('product', this.item, ['Name', 'Image']);
ℹ️autoUpload tự scan entity decorators (DataType.Image/File/Video), emit AutoUpload event, và chờ upload xong trước khi tiếp tục.
