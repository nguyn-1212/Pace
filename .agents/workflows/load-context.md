# Load Context

Trước khi bắt đầu, hãy đọc file context của dự án:

1. Đọc file `.agents/CONTEXT.md` để nắm toàn bộ:
   - Cấu trúc thư mục API và UI
   - Cấu trúc MetaJson hiện tại của QT.01
   - Pattern AttachmentItems, Badge, Tooltip
   - CSS variables của diagram
   - Các shared components

2. Dựa trên nội dung đã đọc, xác nhận với người dùng những gì bạn đã hiểu và bắt đầu task.
